## thinkphp-blogphp

**thinkphp-blogphp**最初开发目的是简单的博客系统，现在已经成长到简单的**CMS企业建站网站系统**。

```
# 克隆
git clone https://github.com/Haiwar/thinkphp-blogphp.git

# 进入项目
cd thinkphp-blogphp

# 安装composer方法
http://blog.musclewiki.cn/post/47?cid=53&index=false

# 安装vendor依赖
composer update
```
